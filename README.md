# kimbi-ms-framework — Кимби СУ, фреймворк системы управления

Абстрактный фреймворк для создания системы управления: понятия, статусы,
связи. К носителю не привязан.

Текст фреймворка — [framework.md](framework.md).

## Версионность

Версия фреймворка указана в тексте (framework.md); релизы отмечаются
git-тегами `vX.Y.Z`. Семантика разрядов:

- **major** — несовместимое изменение модели: у существующих реализаций
  ломается соответствие;
- **minor** — совместимое расширение модели: новая сущность, связь, поле;
- **patch** — формулировки и исправления без изменения модели.
